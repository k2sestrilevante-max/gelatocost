# K2 GitHub Pages Ready

Carica questi file nella root del repository GitHub.

File inclusi:
- `index.html`
- `404.html`
- `.nojekyll`

Dopo il caricamento:
1. apri **Settings > Pages**
2. scegli **Deploy from a branch**
3. branch `main`
4. cartella `/(root)`

Questa versione ├© standalone, senza build e senza dipendenze esterne.
